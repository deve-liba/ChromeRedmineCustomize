Redmine Injector - テストページ

拡張機能テスト

このページは Chrome 拡張機能 "Redmine Injector" のテストに使用します。

テスト手順:
  1. 拡張機能の設定画面を開く
  2. 新規カスタマイズを追加:
    • 名前: テスト用JavaScript
    • 対象ドメイン: localhost (または 127.0.0.1)
    • JavaScript: 下記のコードをコピー
  3. このページをリロード (F5)
  4. 緑色の成功メッセージが表示されることを確認
  5. ブラウザのコンソール (F12) でCSPエラーがないことを確認

テスト用 JavaScript コード:

document.addEventListener('DOMContentLoaded', function() {
  var testResult = document.getElementById('test-result');
  if (testResult) {
    testResult.style.background = '#d4edda';
    testResult.style.border = '2px solid #27ae60';
    testResult.style.color = '#155724';
    testResult.innerHTML = '✓ 成功! カスタムJavaScriptが正常に実行されました。
' +
                          '現在時刻: ' + new Date().toLocaleString('ja-JP');
    console.log('[Redmine Customize Test] JavaScript injection successful!');
  }
});

テスト用 CSS コード (オプション):

body {
  border-top: 5px solid #3498db !important;
}

h1 {
  color: #e74c3c !important;
}
⏳ 待機中... カスタマイズが適用されるとこのメッセージが変わります。

期待される結果

  • ✓ 上記の「待機中...」メッセージが緑色の「成功!」メッセージに変わる
  • ✓ ブラウザコンソールにエラーがない (特にCSPエラー)
  • ✓ CSSを追加した場合、ページ上部に青いボーダーが表示される
  • ✓ CSSを追加した場合、h1タイトルが赤色に変わる

トラブルシューティング

カスタマイズが適用されない場合:

  • 拡張機能が有効になっているか確認
  • ドメイン設定が正しいか確認 (localhost または 127.0.0.1)
  • カスタマイズが「有効」になっているか確認
  • ページをリロード (Ctrl+F5 または Cmd+Shift+R)
  • ブラウザコンソール (F12) でエラーを確認
